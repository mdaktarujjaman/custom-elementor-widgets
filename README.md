# === ITzone360 Widgets ===

Production-ready custom Elementor widgets — 
built with PHP WordPress Plugin API.

## Widgets
- **Info Card** — Icon, title, description, CTA button
  with full style controls (color, typography, shadow)
- **CTA Button** — Advanced call-to-action with hover effects

## Tech Stack
PHP, Elementor Widget API, WordPress Plugin API, CSS3

## Installation
1. Upload to `/wp-content/plugins/`
2. Activate in WordPress Dashboard
3. Open Elementor → search "Info Card" or "CTA Button"
