Custom Elementor widgets with Info Card and CTA Button.

== Description ==

ITzone360 Widgets adds lightweight and customizable Elementor widgets to help you build modern, responsive, and professional websites without writing code.

== Features ==

* Info Card
* CTA Button
* Style Controls
* Responsive
* Lightweight

== Installation ==

1. Upload the plugin to the /wp-content/plugins/ directory.
2. Activate the plugin.
3. Make sure Elementor is installed and activated.
4. Edit any page with Elementor.
5. Drag and drop ITzone360 widgets.
